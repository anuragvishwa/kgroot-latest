VERIFICATION COMMANDS:

1. View all pods:
   kubectl get pods -n {{ .Values.namespace }} -o wide

2. View events:
   kubectl get events -n {{ .Values.namespace }} --sort-by=.lastTimestamp | tail -n 50

3. View logs from a specific scenario:
   kubectl logs -n {{ .Values.namespace }} -l scenario=crashloop --tail=50

4. Check resource usage (requires metrics-server):
   kubectl top pods -n {{ .Values.namespace }}

CLEANUP:

To remove the terminating-stuck pod (if needed):
   kubectl patch pod terminating-stuck -n {{ .Values.namespace }} -p '{"metadata":{"finalizers":null}}'

To uninstall this release:
   helm uninstall {{ .Release.Name }} -n {{ .Values.namespace }}
